Compilando o sistema

Passos para compilar o sistema

1: Verifique se a biblioteca apicartas.jar encontra-se na pasta lib dentro da pasta src
2: Caso a pasta lib contendo a biblioteca apicartas.jar exista, passe para o passo 6
3: Acesse https://github.com/Helisonrt/Baralho, clone o repositório e siga os passos para compilar a biblioteca que se encontra no arquivo README.md que se encontra neste repositório.
4: Após compilar a biblioteca apicartas.jar crie uma pasta chamada lib dentro da pasta src que se encontra nesse projeto
5: Copie o arquivo apicartas.jar para dentro da pasta lib criada no passo anterior.
6: Abra o Terminal e vá até o diretório onde se encontra o arquivo build.xml, provavelmente o mesmo diretório onde se encontra esse README.
