Compilando o instalador

OBS: È necessário que a construção do sistema já tenha sido efetuada, para saber como leia o arquivo README CONSTRUCAO

Passo para compilar o instalador:

1)Abra o terminal e vá até o diretório onde o arquivo instalador.xml se encontra
2)Digite o comando compile instalador.xml e aguarde até que o mesmo seja construido
